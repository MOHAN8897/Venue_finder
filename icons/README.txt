VenueFinder PWA Icon Assets
===========================

This directory contains placeholder icon files for PWA and mobile app support.

Replace these with your branded assets for production!

Required icons:
- icon-192x192.png      (PWA, Android)
- icon-384x384.png      (PWA, Android)
- icon-512x512.png      (PWA, Android, maskable)
- apple-touch-icon.png  (180x180, iOS home screen)
- favicon-32x32.png     (Browser tab)
- favicon-16x16.png     (Browser tab, legacy)
- favicon.ico           (Browser tab, legacy)
- icon.svg              (Optional, modern browsers)
- shortcut-book.png     (96x96, PWA shortcut)
- shortcut-bookings.png (96x96, PWA shortcut)

Best practices:
- At least one icon (512x512) should be maskable (see https://maskable.app/editor)
- All icons should be PNG, except favicon.ico and icon.svg
- Place all icons in this directory and reference them in manifest.json and index.html

To replace: Use your logo, ensure proper padding and background, and optimize for all platforms.
